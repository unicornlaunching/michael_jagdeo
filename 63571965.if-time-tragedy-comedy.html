One of the first things you’ll learn in algebra is:

1 + 1 = 2

The next thing you learn in algebra is that you can rearrange the numbers as follows:

1 = 2 - 1

Consider that:

Time + Tragedy = Comedy

Let’s understand the significance behind this equation by using simple, everyday algebra.

Time + Tragedy = Comedy

Time + Tragedy = Comedy (Buddhism)

therefore

Tragedy = Comedy - Time (Too Soon)

therefore

Comedy = Time + Tragedy (Fool’s Journey)
